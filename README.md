# 极简工具箱合集

一款轻量、无广告、纯净、多工具聚合的个人微信小程序工具箱。

## 功能列表

### 1. 首页
- 网格布局工具图标入口，简洁清爽
- 工具分类：日常工具、计算工具、文本工具、生活工具
- 搜索框：可搜索工具名称
- 下拉刷新、简单动画

### 2. 工具列表（全部前端实现，无需后端）
- **电子木鱼**：支持自动敲击、手动敲击、功德计数、音效和振动反馈
- **亲戚计算器**：支持三代亲戚关系计算，包括爷爷辈和孙子辈，使用穷举法定义固定关系
- **手持弹幕**：支持文本输入、字体大小调整、颜色选择、横屏全屏播放
- **尺子**：支持厘米/英寸切换、数字标出、横屏展示
- **计算器**：标准计算功能，支持加减乘除、小数点、百分比
- **文本大小写转换**：支持大写、小写、标题大小写转换
- **单位换算**：支持长度、重量、温度换算，重量单位包括毫克、克、两、斤、千克、吨、磅、盎司
- **时间戳转换**：支持时间戳和日期时间互转
- **身份证号校验**：验证身份证号格式，提取出生日期、性别等信息
- **颜色拾取器**：选择颜色，支持HSL到HEX转换
- **图片压缩**：前端实现图片压缩功能
- **密码生成器**：支持自定义长度和字符类型的密码生成
- **照片调色**：支持上传照片，选择预设滤镜，调整强度、曝光、对比度等参数，保存到本地
- **分贝测试仪**：模拟界面，显示波形动画
- **随机数生成**：可设置范围的随机数生成

### 3. 我的页面
- 用户头像（默认）
- 收藏常用工具
- 清空缓存
- 使用说明
- 关于页面

## 技术栈
- 微信原生小程序
- 基础库 2.30.0 以上
- 纯前端实现，无后端依赖
- 响应式设计，使用 rpx 布局
- 支持暗色/浅色适配

## 项目结构
```
miniprogram-1/
├── pages/           # 页面文件
│   ├── index/       # 首页
│   ├── mine/        # 我的页面
│   └── tools/       # 工具页面
├── images/          # 图片资源
├── utils/           # 工具类
├── app.js           # 小程序入口
├── app.json         # 全局配置
├── app.wxss         # 全局样式
└── project.config.json # 项目配置
```

## 如何使用
1. 在微信开发者工具中导入项目
2. 选择项目目录：`miniprogram-1`
3. 填写 AppID（如果没有可以使用测试号）
4. 点击「编译」按钮
5. 扫码预览或在模拟器中查看

## 部署说明
1. 在微信开发者工具中点击「上传」按钮
2. 填写版本号和更新说明
3. 点击「上传」
4. 登录微信公众平台，在「版本管理」中提交审核
5. 审核通过后即可发布

## 注意事项
- 所有工具纯前端实现，不请求后端
- 样式统一、简约、现代、暗色/浅色适配
- 全部使用 rpx 布局，兼容手机和平板
- 代码结构清晰，注释清晰、命名规范、无冗余代码
- 无广告、无复杂登录、无权限申请
